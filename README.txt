Консольное приложение для учета студентов, написанное с помощью технологии Spring Boot. Для взаимодействия с приложением используется зависимость Spring Shell, которая позволяет использовать аннотации @ShellComponent - для обозначения, что в классе может быть метод с аннотацией @ShellMethod, @ShellMethod - для обозначения метода, который можно вызвать из консоли. При создании студента создается событие, а затем его публикуем, после чего его перехватывает аннотация @EventListener, которая вызывает необходимый метод. Также используется аннотация @ConditionalOnProperty для создания бина в том случае, когда зависимость в application.properties имеет соответствующее значение. Также мы используем аннотацию @EventListener со значением ApplicationStartedEvent.class, чтобы первым делом проинициализировать repository. Создан Dockerfile для создания контейнера, чтобы запускать это приложение с помощью Decker.
